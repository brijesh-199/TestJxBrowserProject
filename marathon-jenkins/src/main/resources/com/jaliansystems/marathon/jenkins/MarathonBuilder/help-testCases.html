Command Description
<TestCase> Name of the test case(s). Usage: testname1 testname2 .
+<TestSuite> Charatcter '+' followed by name of the test suite. Usage: +suite_name
@<Feature> Charcter '@' followed by name of the Feature. Usage: @feature_name
